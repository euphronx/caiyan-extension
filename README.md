# 猜盐网站速通浏览器扩展

一款帮助快速通过“猜盐”网站挑战的浏览器扩展工具。

## 特性

- 一键选择并自动通关挑战
- 自动打开新标签页并注入解题脚本
- 支持部分挑战的自动速通
- 适用于 Edge / Chrome 浏览器

## 实现原理

1. 用户通过插件 `popup.html` 界面选择要通关的挑战
2. 选择后，`popup.html` 向 `background.js` 发送消息
3. `background.js` 自动打开新标签页，并注入对应的通关脚本
4. 脚本通过 `fetch` 获取挑战答案，并自动填入提交

## 📦 安装使用

### 1. 克隆项目

```bash
git clone https://github.com/euphronx/caiyan-extension.git
```

### 2. 加载扩展

#### Chrome / Edge 浏览器

- 打开浏览器，进入扩展管理页面
  - Chrome：`chrome://extensions/`
  - Edge：`edge://extensions/`
- 开启“开发者模式”
- 点击“加载已解压的扩展程序”
- 选择项目文件夹即可

## 使用说明

1. 点击浏览器工具栏中的扩展图标
2. 在弹出窗口中选择要通过的挑战
3. 扩展会自动打开新标签页并完成挑战
4. 可选择是否要在通关后关闭标签页

## 注意事项

- **浏览器兼容**：仅支持 Chrome 及 Edge 浏览器
- **挑战覆盖**：目前仅支持部分挑战的速通代码，后续会不定期更新
- **登录要求**：“猜病”挑战需要在已登录状态下才能进行
- **网络影响**：受网络环境影响，脚本可能执行失败，重试几次即可
- 使用过程中如遇问题，欢迎提交 [Issue](https://github.com/euphronx/caiyan-extension/issues)
